Selenium Webdriver and C# has been used for this automation solution. Page objects model and Nunit framework have been used.

Folder Structure :

PageObjects : All the domain page objects are stored here
Webdriver : All the methods related to webdriver are written here.
Tests : All the tests are written under this folder
App.config file : The main url of domain is stored here. 

The script will run on Chrome or Firefox browsers. If no browser name is mentioned, if will run on Chrome. The browser to be used can be mentioned in the App.config file.

How to Run :
1. Download the project
2. Open the project in visual studio
3. Open Test > Test Explorer > Click Run all to run the tests.
